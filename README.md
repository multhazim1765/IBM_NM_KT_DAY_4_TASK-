# Library Book Management System

This is a **Library** Book Management System built as part of IBM_TASK_4 using React and Create React App.[file:1]

## Getting Started

### Prerequisites

- Node.js 14 or later installed on your machine.[file:1]
- npm (comes with Node) or Yarn as a package manager.[file:1]

### Installation

1. Clone this repository.
2. Install dependencies:

   ```bash
   npm install
   # or
   yarn
   ```
